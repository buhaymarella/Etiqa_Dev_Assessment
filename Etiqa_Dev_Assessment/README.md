# Etiqa .NET Developer Assessment


## Installation/Running Instructions
  - Clone the repository
  - Restore the database name EtiqaDB.bak located in the Database folder to your local SQL Server instance.
  - Update the connection string in the appsettings.json file to point to your local SQL Server instance.
  - Open the solution in Visual Studio and build the project to restore NuGet packages. 
  - Run the application (dotnet run).
